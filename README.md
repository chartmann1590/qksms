# QKSMS (Fork)

This is a personal fork of the QKSMS application, currently under active development. This version includes updated analytics and integrated AdMob support.

## Download

You can download the latest release, including the signed APK, from the [Releases Page](https://github.com/chartmann1590/qksms/releases/tag/v3.9.4-analytics).

## About

QKSMS is an open source replacement to the stock messaging app on Android. This fork aims to maintain the core experience while adding specific enhancements for analytics and monetization.

## Reporting bugs

When reporting a bug on this fork, please ensure you are using the latest release from this repository.
